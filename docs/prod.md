# Project: Money Hunter (트윈 엔진) - 최종 기획서

## 1. 프로젝트 개요
* **프로젝트명:** Money Hunter (머니 헌터)
* **서비스 브랜드:** 캐치캐치 (Catch Catch)
* **목표:** 개발 착수 2주 내 런칭 / 월 순수익 1,000,000원 달성 (초기)
* **핵심 가치:** "사용자의 시간을 아껴주고, 돈이 되는 정보를 낚아챈다."
* **R&R:**
    * **총괄:** 전체 지휘 및 의사결정
    * **기획(Jay):** 시장 분석, 전략 수립, BM 설계
    * **디자인(Stitch):** UI/UX, 브랜딩, 로고
    * **개발(Antigravity):** 크롤링 엔진, 알림 봇, MCP 연동

## 2. 서비스 구조 (Twin Engine Strategy)

### A. 엔진 1: 캐치 보틀 (Catch Bottle) - Premium Line
* **타겟:** 3040 남성, 위스키/와인 애호가, 리셀러
* **핵심 기능:**
    * 편의점(CU/GS25), 마트(코스트코/트레이더스) 재고 및 입고 알림
    * 데일리샷 등 주류 앱 핫딜 정보 크롤링
* **BM (수익 모델):** 월 구독료 9,900원 (초기 무료 베타 후 전환)
* **디자인 컨셉:** Dark, Gold, Luxury (신뢰감, 프리미엄)

### B. 엔진 2: 캐치 딜 (Catch Deal) - Traffic Line
* **타겟:** 알뜰족, 주부, 대학생, 뽐뿌/펨코 유저
* **핵심 기능:**
    * 주요 커뮤니티(뽐뿌, 루리웹 등) 핫딜/오류 가격 실시간 알림
    * 키워드 필터링 (예: "스탠바이미", "맥북", "특가")
* **BM (수익 모델):** 제휴 마케팅 (쿠팡 파트너스, 알리 어필리에이트) 수수료
* **디자인 컨셉:** Vivid Red/Yellow, Speedy (긴박함, 직관성)

## 3. 기술 요구사항 (For Antigravity)
1.  **Multi-Source Scraper:** Python/Node.js 기반의 비동기 크롤러 구축
2.  **Notification System:** 텔레그램 봇 API / 카카오톡 알림톡 연동 (속도: 게시글 등록 후 1분 내)
3.  **Affiliate Link Converter:** 원본 링크를 파트너스 수익 링크로 자동 변환하는 로직 탑재

## 4. 디자인 요구사항 (For Stitch)
1.  **BI (Logo):** '캐치캐치' 통합 로고 및 엔진별 서브 로고
2.  **Landing Page:** 원페이지 스크롤 방식의 소개 및 구독 설정 페이지 (모바일 최적화)
3.  **MCP Server:** Antigravity가 호출할 수 있는 디자인 리소스(Color, Font, Asset) 제공

## 5. 마케팅 전략
* **SEO:** "검색 최적화"를 통한 자동 유입 (블로그 자동 포스팅)
* **Viral:** 초기 "개발자 스토리"를 활용한 커뮤니티 침투 (진정성 호소)
